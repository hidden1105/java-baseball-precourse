# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
###요구사항 1
* 입력받기 & 기본 흐름
  * 3자리 숫자
  * 서로 같은 숫자
  * 1~9범위 숫자  
* 볼계산
* 스트라이크계산
* 결과출력(스트라이크, 볼, 낫싱)
* 반복
* 재시작 or 종료
* 랜덤 정답 만들기
* 잘못입력해도 에러 출력 후 진행

###요구사항 2
* indent 확인
* else 금지
* 함수 10라인 이상 금지 (한 가지 일만 하도록)

###요구사항 3
* 핵심 로직과 UI담당하는 로직 분리 구현

### 리팩토링
* 클래스 및 함수 분리가능한지 확인
* 기타 (가독성 등)
